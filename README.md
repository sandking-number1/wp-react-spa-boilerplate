# wp-react-spa-boilerplate

> Theme boilerplate for WordPress using WP REST API and React.js

## Contains

- [x] Babel
- [x] BrowserSync
- [ ] CSS Modules or CSS in JS
- [x] React
- [x] React Routor
- [x] React Hot Module Reload
- [x] Standard
- [x] Wordpress 4.7+
- [x] webpack
- [x] yarn

## Setup

first time only.

```
$ yarn run setup
```

Development

```
$ yarn run develop
```

Production

```
$ yarn run production
```
